# Problem:

(From CodeSignal)

Given an array of integers `a`. A new array `b` is generated by rearranging the elements of `a` in the following way...

- b[0] is equal to a[0]

- b[1] is equal to the last element of `a`

- b[2] is equal to a[1]

- b[3] is equal to the second-last element of `a`

- and so on...

The task is to determine whether the new array b is sorted in strictly ascending order or not.

Example 1:

Input: a:[1, 3, 5, 6, 4, 2]

Output: true

Example 2:

Input: a:[1, 3, 5, 6, 4, 20]

Output: false

Example 3:

Input: a = [1, 3, 5, 6, 40, 2]

Output: false

# Solution:

{% highlight js %}
(function main() {
    const a = [1, 3, 5, 6, 40, 2];
    let b = [];
    let i = 0;
    let j = a.length;

    for (i=0; i<j; i++){
        if (i == 0){
            b[0] = a[0];
        }
        else if (i % 2 == 0){
            b[i] = a[i/2]
        }
        else {
            b[i] = a[j-((i+1)/2)]
        }
    }
    for (i=0; i<b.length; i++){
        if (b[i]>b[i+1]){
            console.log("false: "+ b);
            return b;
        }
    }

    console.log("true: "+b);

}());
{% endhighlight %}
